3.ip代理：
为避免同一ip重复刷量被监测的情况，必须设置ip代理，ip代理的跳板当前80%以上需为云南地区，（建议购买可用且稳定的代理服务器，ip代理数量尽可能多），代码实现动态切换代理，随机切换ip进行刷量。
4.Wt.event 和 wt.es 字段：
建议最好支持可配置，在每一次开始刷量之间，根据运营提供的触发量大的页面对wt.event 和 wt.es进行随机切换（如wt.event='banner位名称'&wt.es='本地人'）。
